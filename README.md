## รายละเอียดเนื้อหาแต่ละหน้าเว็บ

### 1. index.html
- Header ชื่อบริษัท  
- Navigation menu  
- Hero section  
- Featured services  
- Footer  

### 2. about.html
- เรื่องราวบริษัท  
- ทีมงาน (ใช้ <figure>)  
- Vision / Mission  
- ปุ่มกลับหน้าแรก  

### 3. services.html
- รายการบริการ  
- <section> ต่อหนึ่งบริการ  
- ตารางเปรียบเทียบแพ็กเกจ  

### 4. contact.html
- Contact Form  
- ที่อยู่บริษัท  
- แผนที่ Google Maps
